📖 What is JSON?

JSON (JavaScript Object Notation) is a lightweight data-interchange format. It's human-readable and easy for machines to parse. JSON supports strings, numbers, booleans, null, arrays and objects.

{"string":"hello","number":42,"bool":true,"null":null,"array":[1,2,3],"object":{"nested":"value"}}

⚠️ Common JSON Errors

Trailing commas (not allowed in JSON), single-quoted strings (must use double quotes), unquoted keys, and comments (JSON doesn't support them) are the most frequent mistakes. This tool highlights the exact error position.
